Jaringan Komputer & Internet

Media pembelajaran mengenai dasar jaringan komputer dan internet untuk pemula.

Materi Pembelajaran

1. Pengertian Jaringan Komputer

Jaringan komputer adalah sekumpulan komputer yang saling terhubung dan dapat saling berbagi data, informasi, dan sumber daya.

2. Tujuan dan Manfaat Jaringan Komputer

  • Berbagi file dan data
  • Komunikasi lebih cepat
  • Berbagi sumber daya (printer, internet, dll.)
  • Meningkatkan efisiensi kerja

3. Jenis-Jenis Jaringan

  • LAN - Local Area Network
  • MAN - Metropolitan Area Network
  • WAN - Wide Area Network
  • PAN - Personal Area Network

4. Topologi Jaringan

  • Bus
  • Star
  • Ring
  • Mesh

5. Perangkat Jaringan

  • Router
  • Switch
  • Hub
  • Modem
  • Access Point

6. Pengertian Internet & Cara Kerjanya

Internet adalah jaringan luas yang menghubungkan berjuta-juta perangkat di seluruh dunia. Internet bekerja dengan sistem protokol TCP/IP.

7. Protokol Internet

  • TCP/IP
  • HTTP / HTTPS
  • FTP
  • DNS
